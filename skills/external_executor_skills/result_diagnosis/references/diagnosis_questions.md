# Result Diagnosis Questions

Answer these from raw results and logs:

1. Which baseline is strongest?
2. Why is that baseline strong?
3. Where does ours work?
4. Where does ours fail?
5. Which module likely caused improvement?
6. Which module is inactive or harmful?
7. Is improvement from the core mechanism, capacity, training trick, or data processing?
8. Should the mechanism be adjusted?
9. Should the claim be narrowed?
10. Is there a bridge or theory insight for T8?

Classify each finding as raw-result backed, diagnostic hint, unsupported, or
requires rerun.
